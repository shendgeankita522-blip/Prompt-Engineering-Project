# Prompt-Engineering-Project
# 🤖 Prompt Engineering using Gemini

A Python-based project that demonstrates different **Prompt Engineering techniques** using **Google Gemini AI**. This project contains multiple prompt examples to understand how different prompting strategies improve the quality of AI-generated responses.

---

## 📌 Project Overview

Prompt Engineering is the process of designing effective prompts to obtain accurate, relevant, and high-quality responses from Large Language Models (LLMs).

This project showcases various prompting techniques with practical Python examples using the Google Gemini API.

## 📦 Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Prompt-Engineering.git
```

Move into the project directory

```bash
cd Prompt-Engineering
```

Install required packages

```bash
pip install google-generativeai
```

---

## 🔑 Configure API Key

Obtain your Gemini API Key from **Google AI Studio**.

Replace your API key inside the Python file.

Example:

```python
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")
```

---

## ▶️ Run the Project

Run any Python example:

```bash
python 1.py
```

or

```bash
python App.py
```
